# Working Knowledge — Document Catalogue

Reference materials for people actually doing the work — code books, compliance guides, technical field references, SOPs. Not training manuals or beginner's guides.

**Spawn philosophy:** These appear in filing cabinets, office desks, lockers, and back-room storage — not on bookstore shelves or book display racks. A gun store's counter drawer might have a firearms compliance reference; the display rack wouldn't. Each entry notes likely container keys; verify room context against game distribution files before implementing.

Multiple options are listed per skill. Spawn keys are noted where confirmed against game distribution files.

### Trapping

*Licensed trapping in Kentucky requires demonstrated knowledge of regulations, species identification, and legal set construction. The KDFWR documentation that supported that licensing process is exactly the kind of reference that stays in a filing cabinet until someone needs it.*

- **Wildlife Management Trapping Field Guide** — A reference issued by the Kentucky Department of Fish and Wildlife Resources. Trap types, placement strategy, and set construction by target species. Keys: `FilingCabinetGeneric` (huntingstore room)
- **Furbearer Identification and Harvest Reference** — A commercial trapping reference covering pelt quality assessment, prime time by species, and handling procedures. Keys: `FilingCabinetGeneric` (huntingstore room)
- **KDFWR Trapper Regulations and Compliance Reference** — Legal set requirements, species restrictions, and tagging requirements for licensed trappers in Kentucky. Keys: `FilingCabinetGeneric` (office room)
